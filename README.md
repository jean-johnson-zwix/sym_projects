# SYM Projects

Internal tooling and automation projects for **Speak Your Mind (SYM)** — a mental health and emotional wellness community for men.

---

## Projects

### [`csm_test_automation`](./csm_test_automation)
Automated testing agent for the SYM Content Manager System (CMS) Telegram bot. Uses Telethon to simulate a real user conversation and Gemini AI to read bot responses, make approval decisions, and score output quality. Produces a session log ready for report generation.
